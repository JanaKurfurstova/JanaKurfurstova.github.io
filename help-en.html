This portal allows searching the catalogues of all participating libraries using single search box. Being registered in one or more participating libraries, it is possible to use services of these libraries via this portal.

Tooltips and Context-Sensitive Help

To get to the specific user interface elements, it is not necessary to go through this documentation.

By clicking on a grey question mark icon, help to a particular issue appears.

Other tooltips appear once you stop the cursor on the title you want to be explained.

1 Signing-in and overview of services according to user permission

1.1 Signing-in

You can log in with an account of any institution listed in "Connect with library's account" or social network account from "Alternative login" list (Google, Facebook, LinkedIn, GitHub, mojeID, ORCID).

Once you sign in with an account of any participating library, you gain access an overview of your activities in this library, and you will be able to order or reserve publications.

When logging in with a particular library account for the first time, the message about transmitted service may appear (the reappearance of this message at next logon depends on how you set the validity of consent). Then you will see a request for consent to storing data in accordance with Act no. 101/2000 Coll. on personal data protection. You will not log in without acknowledgment and agreement to these two questions.

In case you are not registered in any of the connected libraries and logged in through a social network account, mojeID or ORCID, you will not be able to order or reserve a publication in connected libraries but you can still save your favorite records and searches.

Learn more about registration in participating libraries here.

You can link your library and social network accounts in Knihovny.cz (read more) - you can use any of them to log in.

2 Search

The search box, working similarly to Google search, is used for simple search.

If the search box is empty, by clicking on "Find" button, the list of all entries within the portal will be shown

The search results can be listed in accordance with a relevance, the newest first, the oldest first, or in alphabetical order according to an author or a title.

2.1 Tips for searching with examples

You can use Boolean operators and wildcards in simple and advanced search.

"Poe's Raven"

  • search phrase between quotation marks, i. e.  Poe's Raven

Poe AND Raven

  • search terms containing  Poe  as well as  Raven

Poe OR Raven

  • search terms containing at least one of the words  Poe  or  Raven

Poe NOT Raven  /  Poe -Raven

  • search terms containing  Poe  and not containing  Raven

+Poe Raven

  • search terms possibly containing  Poe  and necessarily containing  Raven

R*ven

  • search terms begining with  R  and ending with  ven  with empty or random string of characters in place of  *  (you cannot use that in the beginning of the word)

R?ven

  • search terms begining with  R  and ending with  ven  with one character in place of  ?  (you cannot use that in the beginning of the word)

Raven~  /  Raven~0.8

  • search terms similar to  Raven  (Haven, Ravel); degree of similarity is defined by setting  value from 0 to 1 (default is 0.5)

"Poe Raven"~3

  • search terms with at most three words between  Poe  and  Raven

Poe Raven^8 poetry^6

  • preferential search where the word significanceis given by the value after  ^  (a term without ^ has the lowest value)

{Po TO Pr}  /  [2001 TO 2003]

  • range search (terms begining with Po, Pp, Pq and Pr / numbers from 2001 to 2003)

2.2 Prompt

To make search easier, a prompt is available in the search box for simple search. This offers the most relevant phrases for the term entered within three categories: titles, authors and topics. User can pick a phrase suggested by the prompt or search the term without use of any prompt.

It is necessary to note that as the portal includes records from many libraries, the same topic can be often found in the prompt more than once. For example, name Karel IV. can be suggested in a number of versions while each of these contain different records.

2.3 Advanced search

A form for advanced search allows users to enter a complex and complicated queries.

For each word it is possible to specify the category of search. There are following fields:

  • In All Fields
  • Title, series
  • Author, corporation and action
  • Subject, key word
  • Published in - for articles
  • ISBN/ISSN/ISMN
  • Call number - signature
  • Publisher
  • Year of Publication
  • Place of Publication
  • ID (001) - record ID provided by libraries in field 001
  • Czech National Bibliography Number - ČNB
  • Fulltexts and contents - of indexed sources

2.3.1 Search box

User chooses the category which will be searched from the drop-down list (with pre-set search "In All Fields").

It is possible to add additional search boxes and enter into them additional search terms.

Logical operators from a short drop-down menu can be applied on any search box in the group. There are following options:

  • All terms - i.e. logical operator AND
  • Any term - i.e. logical operator OR
  • No term - i.e. logical operator NOT

2.3.2 Search group

Adding a search group allows the user to add additional search boxes, on which another logical operator can be applied. Therefore, for example records in which occur all terms from the first group while only selected terms from another group can be searched.

Logical operator can be applied also on search groups. There are following options:

  • All groups - i.e. logical operator AND
  • Any groups - i.e. logical operator OR

To apply the changes in the form it is necessary to "Update results".

To clear the form, click on the "Clear".

Příklad: Search the publications about Daniel Defoe not written by Daniel Defoe.

  1. Type “Daniel Defoe” to the search box with preset searching “In all fields”
  2. Use “Add search box” button under the search form
  3. In the dropdown menu of the new group choose “No term”
  4. Choose the search option “Author” for the new searchbox and type “Daniel Defoe”
  5. Click on the “Update results” button to apply your search query

3 Displaying the search results

The number of results per page can be set in the drop-down menu in "View per page".

Search results can be also arranged through "Sort" in the drop-down menu:

  • by most relevant - i.e. results, which correspond the most with the search query (pre-set)
  • by newest
  • by oldest
  • by author
  • by title

Besides basic information about a publication, number of institutions where the publication is available is displayed.

This icon  is visible next to the name and year of birth/death of some authors. The icon is a link to the detailed information about the author, i.e. the authority record.

3.1 Filters ("Narrow search")

You can use filters to make search results more accurate.

When using filter "Institution" and "Year of publication” confirm your choice by "Use filter" or "Use" button .

The use of icons in filter "Institution":

 - save a selection for later use. This icon is displayed only to the logged-in user. Saved selection can be reset by unchecking all the selected libraries/resources and clicking the icon again.

. - load your saved selection. This icon is displayed only to the logged-in user.

 - select libraries where you are registered. This icon is displayed only to the logged-in user.

 - find the nearest libraries.

You can remove applied filters one by one (the pink cross pictogram ) or all at once ("Remove filters" ).

If you are using any filter and you want to enter a new search query, you can choose the option to “Search with current filters” (checkbox under the search field). This option is set as default and can be unchecked.

3.2 Save, send and track (RSS) the search results

At the top area of the page, right of the search box. there is "Save search" button. Your saved searches are available in section "My account - "Saved searches".

You can send the search results via email by clicking the envelope icon at the end of the search results list.

The RSS icon at the end of the search results list refers to the RSS feed of the search results.

You can track the latest publications in your field by subscribing to the search results RSS feed

Recommended steps before adding the feed to your RSS reader:

  1. Sort search results by newest. 
  2. Specify your subject of interest precisely using filters and advanced search. 
  3. Set “Views per page” according to publication acitivity in your field. 

Example: Are you interested in new books on zoology available to outside loan in Moravian Library?

4 Displaying the full record

The full record shows bibliographic record of publication and the availability of copies (library units) in a particular library.

Record is available for view from each library having the publication listed in its catalogue.

It is possible to switch among records of different libraries by drop-down list "Select instituion". To see opening hours, contact or address, click on "Information about library".

If the user is signed-in with an account connected to one or more libraries where they are registered, first records shown would be from those libraries.

This icon  is visible next to the name and year of birth/death of some authors. The icon is a link to the detailed information about the author, i.e. the authority record.

4.1 Side panel options

Bibliographic record has an option "Email this".

It is also possible to "Export record to reference managers or to download it in .txt format.

"Permalink" leads directly to the displayed record. Inserted anywhere, it will still refer to the record as it is displayed in Knihovny.cz.

Another option for publication is the "Add to favorites". User’s favorite records are available in section "My account" - "Favorites".

Feature "Show citations" offers correct bibliographic entry according to the chosen standard.

Column "Similar" displays publications with similar record to the current one. It is possible that here the same publication appears in case this record have not merged with others. As records of libraries are very diverse, it is problematic to eliminate all of these. However, we are still working on optimization of deduplication algorithms.

4.2 Tab "Borrow"

A list of units of a particular publication, which is available in library. It is possible to switch among libraries in the drop-down list "Select institution".

If the user is signed-in by an account connected to a library where he is registered, a button "I want to borrow" can appear next to some of publication units.

Button "I want to borrow opens a simple form, which allows the user to reserve the unit, which is at that time borrowed by another user, or it allows to order the publication from stock, or ask the unit to be hold at the particular branch library.

Conditions or borrowing, ordering and reservations are different for each library. Every library can offer its own help for the "Borrow" card as well as for the order form. Help can be opened by clicking on the grey question mark icon.

4.3 Tab "E-version"

This tab offers a list of electronic versions of document in all libraries, which have this document available in digital variant.

  • Available documents can be read from anywhere.
  • Copyrighted documents are available only on computers in the library.
  • Unknown availability of the document about which the information on availability is missing, is possible on the library’s website.

4.4 Tab "Comments"

A place for reader’s ratings and opinions. Signed-in users can add their comments here.

4.5 Tab "Metadata"

Contains the publication record in MARC 21 format, which is relevant for librarians.

5. Your Account

User gains access to various features by signing in to portal Knihovny.cz. The list of these is available in the first chapter of this manual.

It is necessary to have already existent account for signing in to Knihovny.cz. It is possible to sign in with any institution listed in "Connect with library's account" or social network account from "Alternative login" list (Google, Facebook, LinkedIn, GitHub, mojeID, ORCID).

One of the most significant advantages for signed-in users is the possibility to link accounts (explained in detail in chapter  Linked accounts).

The portal itself does not allow to create a new account. However, it is possible to register online, to pre-register or to register remotely in some of participating libraries. The list of libraries is available here. Pre-registration differs from registration by the requirement to visit a library in person to finish the registration (usually to pay the registration fee). Remote registration consists of more steps than online registration (according to library’s instructions).

Information described in the following chapters are available after signing in. It can be found in the menu after clicking on the user name in the upper right corner of the screen.

5.1 Checked Out Items

Your current loans in connected libraries.

You can try to postpone your due date if "Renew" button is visible.

Renewing the loan is probably not possible in following cases:

  1. Some other user has the book reserved.
  2. You have already renewed your loan before and reached the limit for renewing particular book.
  3. Your library card / registration expires soon.
  4. You have some fine or charge to pay.

This overview is visible only to users with linked account of at least one of the connected libraries (read more).

5.2 Checked out history

This section shows a list of publications, which have been borrowed by user. Not all of participating libraries may support this feature on Knihovny.cz

This overview is visible only to users with linked account of at least one of the connected libraries ( read more).

5.3 Orders and Reservations

Your current orders and reservations in connected libraries. 

The option to cancel your orders and reservations could be time limited depending on system settings of your library. Orders could be usually canceled just shortly after ordering, if even possible to do so, if at all. 

If the publication you request has been borrowed by other user, you can make a reservation. 

Ordering is usually used for publications stored in a library closed stack, or for putting on hold in branch library.

Conditions for borrowing, ordering and reservations are different for each library - some of the connected libraries provide a custom help visible after clicking on the grey question mark in the request form screen.

This overview is visible only to users with linked account of at least one of the connected libraries (read more).

5.4 Fines and Charges

Overview of fines and charges to be paid, amount of deposit in reader’s account.

Outstanding fines and charges are displayed as negative amount.

Your reader's account deposit used for covering future charges are displayed as positive amount.

It is possible that a library does not allow readers in debt to use some online services (renew loans, holds, recall requests…).

Some libraries allow their readers to pay fines and charges via portal Knihovny.cz. In this case "Pay fines online" or "Pay charges online" button appears and  redirects user to library’s payment gateway.

This overvieiw is visible only to users with linked account of at least one of the connected libraries (read more).

5.5 Profile

Your membership informationin the connected libraries - changes are possible while visiting library in person or on the library’s website (according to the rules of particular library). Profile information cannot be changed using portal Knihovny.cz.

Knihovny.cz itself allows logging in with already existing library or social network account and follows the policy of storing the minimal amount of user data. Czech academic identity federation eduID.cz allows connecting user accounts of participating libraries. Every library involved in Knihovny.cz project has to follow security policy set by eduId.cz.

Some libraries allows you to prolong the validity of your library card through Knihovny.cz portal. In this case the "Prolong registration online" button is visible. Clicking the button redirects you to the library's payment gateway.

This overvieiw is visible only to users with linked account of at least one of the connected libraries (read more).

5.6 Settings

Permanently preferred format of citations, number of results per page and the type sorting the results according to their relevance, number of borrowings, date, author or title can be set here.

5.7 Linked Accounts

Link your accounts from connected libraries and social networks.

SAFETY WARNING: Linking accounts on public or shared computers is strongly discouraged for safety reasons!

 

Click on "Connect another account" button and continue by logging in with an account of any institution listed in "Connect with library account" or social network account from "Alternative login" list (Google, Facebook, LinkedIn, GitHub, mojeID, ORCID).

Once you attach other connected library's account to your identity(*), you gain access an overview of your activities in this library, and you will be able to order or reserve publications.

In case you are not registered in any of the connected libraries and logged in just using a social network account or mojeID or ORCID, you will not be able to order or reserve a publication in connected libraries, though you can still save your favorite searches and records.

Learn more about registration in participating libraries here.

You can use any of your connected accounts to access your idenity. You can simply disconnect your linked accounts anytime. It is also possible to "Delete identity" - this action removes all your user settings, saved searches and favorites lists

We recommend to link accounts using your own computer.
In case you really need to link your acconts na publicly accessible PC, make sure that before leaving you have deleted all cookies in the browser (for this portal as well as for other institutions used for linking your accounts).

Linking more than one account within one library is not possible. It is also not possible to link one account from library A with more than one account from library B (as library cards are not transferable).

(*) The identity in Knihovny.cz ties your linked library and social network accounts. When you sing in for the first time, your portal identity includes just one account - that one you used for your first logging in. The decision to connect more accounts is up to you.

5.8 Saved searches

User’s recent and saved searches.

Any of the recent searches can be saved for later use. In case it is no longer needed, any saved search can be removed.

Any unsaved searched can be removed by clicking on "Remove an unsaved search".

The most recent searches are available also for users that are not signed in, from a link in the footer of the left column.

5.9 Favorites

All records you marked as favorite during sessions. 

You favorite records can be sorted into the custom lists and notes can be attached to them (use the “Edit” button). 

The items you marked as favorite as an unlogged user, are saved in the “Transferred favorites” list. You can also rename your lists and describe them using the “Edit list” button. 

Other features:

  • Delete
  • Email: If only one item was selected, the recipient receives a link to the full record. If more than one item is selected, a link to the whole list is sent - the recipient will not see the list name, or any of your other user data.
  • Export: Download bibliographic citations of selected items as a plain text file or export them to one of the listed reference managers.
  • Print

6 Inspiration

Inspiration shelf offers the readers an interesting selection of fiction and scientific literature. Currently it includes recommendation e.g. for books, which were made into a movie or TV adaptations, the most borrowed books from the previous years, the most popular authors from a range of genres, or an offer of electronic documents available for download. We gladly accept any recommendations for more such lists from libraries‘ employees.

There is also the hierarchical list of disciplines - each of its item refers to a list of publications related to the subject.

7 Address book of Czech libraries ("Libraries")

The Czech Republic has the densest network of public libraries in whole European Union.

This section offers a  contact information including opening hours of all libraries, which are listed in the directory of National Library of the Czech Republic. As the directory data are retrieved from National Library’s Central Directory of libraries, it is possible that for some libraries, the branch libraries are listed separately (for these have separate SIGLA). Data changed in CADR will not appear in the directory immediately, but after a few hours.

Majority of these libraries do not participate in portal Knihovny.cz. Nonetheless, the number of participating libraries will gradually increase.

To find the library it is possible enter a query to the search box (name, city/town, or SIGLA can be entered), or to find it on the map.
