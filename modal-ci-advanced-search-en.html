A form for advanced search allows users to enter a complex and complicated queries.

Terms entered to the form for advanced search can contain the same operators and wildcards as in basic search.

Search boxes

User chooses the category which will be searched from the drop-down list (with pre-set search "In All Fields (without full text)").
It is possible to add additional search boxes and enter into them additional search terms.
Logical operators from a short drop-down menu can be applied on any search box in the group. There are following options:

  • All terms - i.e. logical operator AND
  • Any term - i.e. logical operator OR
  • No term - i.e. logical operator NOT

Search group

Adding a search group allows the user to add additional search boxes, on which another logical operator can be applied. Therefore, for example records in which occur all terms from the first group while only selected terms from another group can be searched.
Logical operator can be applied also on search groups. There are following options:

  • All groups - i.e. logical operator AND
  • Any groups - i.e. logical operator OR

To apply the changes in the form it is necessary to "Update results".
To clear the form, click on the "Clear".
